텍스트 처리 서비스¶

이 장에서 설명하는 모듈은 광범위한 문자열 조작 연산과 기타 텍스트 처리 서비스를 제공합니다.

바이너리 데이터 서비스 에 기술되어 있는 codecs 모듈 또한 텍스트 처리와 밀접한 관련이 있습니다. 또한, 텍스트 시퀀스 형 --- str 에 있는 파이썬의 내장 문자열형에 대한 설명서를 참조하십시오.
